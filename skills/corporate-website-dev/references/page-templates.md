# Page Templates Cheat Sheet

Summaries of common corporate-site pages. Use these as checklists when translating requirements into sections, copy blocks, and components.

## 首页 / Hero
- Fold: punchy headline, 1-line value prop tied to industry insight, CTA pair (primary action + secondary learn-more)
- Visual: hero illustration, product UI mock, or ambient video background (optimize for <2MB poster image)
- Credibility: client logos, quick KPI strip (e.g., "10K+ enterprise users")

## 关于我们
- Narrative arc: origin story, mission/vision, leadership highlight
- Timeline or milestones component for traction proof
- Culture values cards if recruiting is in scope

## 产品 / 解决方案
- Start with pain → solution framing
- Feature grid (3–6 cards) with iconography tied to UI style guide
- Technical architecture diagram or bullet stack for trust with engineering buyers

## 功能亮点 / 价值主张
- Pillar layout: statement, supporting metric, micro-CTA linking to detailed doc/demo
- Optional animated stepper for workflows

## 定价
- Tier cards (3 typical). Include plan name, ideal customer, price (or "联系我们"), bullet benefits, CTA
- Footnotes for billing cadence, SLAs, compliance badges

## 成功案例 / 客户见证
- Case-study cards with logo, problem → outcome narrative, quantified impact
- Quote carousel (include headshot + title if assets exist)

## 博客 / 洞察
- Filter bar (topic/category)
- Cards showing hero image, publish date, estimated read time
- CTA for newsletter subscription

## 资源 / 下载中心
- Tabbed sections by asset type (whitepaper, report, webinar)
- Metadata chips: format, size, gated/ungated indicator

## 招聘 / 加入我们
- EVP statement + culture proof points
- Role list (integrate ATS links) grouped by department
- Benefits comparison table

## 联系我们
- High-emphasis CTA block (email, phone, WeChat/Slack community)
- Embedded form (name, work email, company, intent)
- Map or regional offices component if relevant

## FAQ
- Accordions grouped by persona/topic
- Link each answer to supporting docs or contact CTA

## 自定义页面
- Start with the same triad: objective, audience, success metric
- Mirror component inventory from similar templates above to reduce net-new work
