### HTML Example for Z-index Utilities

Source: https://getbootstrap.com/docs/5.3/utilities/z-index

Demonstrates how to use Bootstrap's z-index utility classes to stack elements. These utilities require a non-static position, which can be set using custom styles or Bootstrap's position utilities. The example shows elements with z-index values from -1 to 3.

```html
<div class="z-3 position-absolute p-5 rounded-3"><span>z-3</span></div>
<div class="z-2 position-absolute p-5 rounded-3"><span>z-2</span></div>
<div class="z-1 position-absolute p-5 rounded-3"><span>z-1</span></div>
<div class="z-0 position-absolute p-5 rounded-3"><span>z-0</span></div>
<div class="z-n1 position-absolute p-5 rounded-3"><span>z-n1</span></div>
```
