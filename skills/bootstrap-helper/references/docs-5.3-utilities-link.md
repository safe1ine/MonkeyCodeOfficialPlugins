### Link Underline Hover Variants (HTML)

Source: https://getbootstrap.com/docs/5.3/utilities/link

This HTML example demonstrates combining Bootstrap's link utilities to create custom hover effects for underlines. It showcases how to mix offset and opacity on hover for unique link appearances. These utilities are purely CSS-driven.

```html
<a class="link-offset-2 link-offset-3-hover link-underline link-underline-opacity-0 link-underline-opacity-75-hover" href="#">
  Underline opacity 0
</a>
```
