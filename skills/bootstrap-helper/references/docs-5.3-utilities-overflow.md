### Bootstrap Overflow Utilities (HTML)

Source: https://getbootstrap.com/docs/5.3/utilities/overflow

These HTML examples demonstrate the use of Bootstrap's general overflow utility classes. They are applied to elements with defined width and height to showcase different overflow behaviors like auto, hidden, visible, and scroll.

```html
<div class="overflow-auto">...</div>
<div class="overflow-hidden">...</div>
<div class="overflow-visible">...</div>
<div class="overflow-scroll">...</div>
```
