### Bootstrap Vertical Alignment CSS Classes for Table Cells

Source: https://getbootstrap.com/docs/5.3/utilities/vertical-align

Illustrates how to apply Bootstrap's vertical alignment utility classes to table cells (`<td>`). This example shows alignment within a table with a fixed height, utilizing classes like `.align-baseline`, `.align-top`, and others to control the vertical position of content within the cells.

```html
<table style="height: 100px;">
  <tbody>
    <tr>
      <td class="align-baseline">baseline</td>
      <td class="align-top">top</td>
      <td class="align-middle">middle</td>
      <td class="align-bottom">bottom</td>
      <td class="align-text-top">text-top</td>
      <td class="align-text-bottom">text-bottom</td>
    </tr>
  </tbody>
</table>
```
