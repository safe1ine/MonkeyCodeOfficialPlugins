### Aligned Figure Caption (HTML)

Source: https://getbootstrap.com/docs/5.3/content/figures

Demonstrates how to align a figure's caption using Bootstrap's text utility classes. This example right-aligns the caption.

```html
<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption text-end">A caption for the above image.</figcaption>
</figure>
```
