### Apply Opacity Utilities in HTML

Source: https://getbootstrap.com/docs/5.3/utilities/opacity

Demonstrates how to apply Bootstrap's opacity utility classes to HTML elements to control their transparency. These classes directly map to opacity values from 0 to 1.

```html
<div class="opacity-100">...</div>
<div class="opacity-75">...</div>
<div class="opacity-50">...</div>
<div class="opacity-25">...</div>
<div class="opacity-0">...</div>
```
