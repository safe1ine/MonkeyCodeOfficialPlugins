### Responsive Object Fit Utilities (HTML)

Source: https://getbootstrap.com/docs/5.3/utilities/object-fit

Utilize responsive variations of object-fit utilities to adjust the resizing behavior of images based on screen breakpoints. Classes are appended with breakpoint abbreviations like 'sm', 'md', 'lg', 'xl', and 'xxl'.

```html
<img src="..." class="object-fit-sm-contain border rounded" alt="...">
<img src="..." class="object-fit-md-contain border rounded" alt="...">
<img src="..." class="object-fit-lg-contain border rounded" alt="...">
<img src="..." class="object-fit-xl-contain border rounded" alt="...">
<img src="..." class="object-fit-xxl-contain border rounded" alt="...">
```

### Object Fit Utilities for Video (HTML)

Source: https://getbootstrap.com/docs/5.3/utilities/object-fit

Apply object-fit utilities to video elements to control how the video content resizes within its container. Both basic and responsive utility classes can be used.

```html
<video src="..." class="object-fit-contain" autoplay></video>
<video src="..." class="object-fit-cover" autoplay></video>
<video src="..." class="object-fit-fill" autoplay></video>
<video src="..." class="object-fit-scale" autoplay></video>
<video src="..." class="object-fit-none" autoplay></video>
```
