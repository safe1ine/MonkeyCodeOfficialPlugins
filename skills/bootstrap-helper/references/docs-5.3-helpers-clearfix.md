### Bootstrap Clearfix Example with Floated Buttons

Source: https://getbootstrap.com/docs/5.3/helpers/clearfix

This example demonstrates how the clearfix utility, combined with Bootstrap's `float-start` and `float-end` classes, correctly contains floated buttons within a background-colored div.

```html
<div class="bg-info clearfix">
  <button type="button" class="btn btn-secondary float-start">Example Button floated left</button>
  <button type="button" class="btn btn-secondary float-end">Example Button floated right</button>
</div>
```
