### Bootstrap CSS Variables for Page Styles

Source: https://getbootstrap.com/docs/5.3/customize/css-variables

Demonstrates how to use Bootstrap's CSS variables to customize global page styles like font and link colors. These variables offer runtime customization without CSS compilation.

```css
body {
  font: 1rem/1.5 var(--bs-font-sans-serif);
}
a {
  color: var(--bs-blue);
}

```
