### Bootstrap: Apply Background Opacity with Utility Classes

Source: https://getbootstrap.com/docs/5.3/utilities/background

Shows how to apply different levels of background opacity using Bootstrap's built-in `.bg-opacity` utility classes. These classes provide predefined opacity levels (75%, 50%, 25%, 10%) for elements with the 'bg-success' class.

```html
<div class="bg-success p-2 text-white">This is default success background</div>
<div class="bg-success p-2 text-white bg-opacity-75">This is 75% opacity success background</div>
<div class="bg-success p-2 text-dark bg-opacity-50">This is 50% opacity success background</div>
<div class="bg-success p-2 text-dark bg-opacity-25">This is 25% opacity success background</div>
<div class="bg-success p-2 text-dark bg-opacity-10">This is 10% opacity success background</div>
```
