### Sass Example: Using Color Variables

Source: https://getbootstrap.com/docs/5.3/customize/color

Demonstrates how to use Bootstrap's Sass color variables and map values to set text and background colors in CSS. It shows direct variable usage and accessing map values.

```scss
.alpha { color: $purple; }
.beta {
  color: $yellow-300;
  background-color: $indigo-900;
}

```

### Apply Subtle Primary Background and Border with Text Emphasis

Source: https://getbootstrap.com/docs/5.3/customize/color

This example demonstrates how to use Bootstrap's utility classes to apply a subtle primary background, a primary border, and primary text emphasis to a div element. It also includes rounded corners for a visually distinct component. The utilities leverage the new color variables introduced in Bootstrap v5.3.0.

```html
<div class="p-3 text-primary-emphasis bg-primary-subtle border border-primary-subtle rounded-3">
  Example element with utilities
</div>
```
